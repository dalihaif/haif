# 2026-05-28 工作日志

## 个人网站开发

为李海峰（大理大学第一附属医院）开发了完整的个人网站。

### 技术栈
- 纯前端静态站（HTML5 + CSS3 + 原生JS）
- 无依赖，零构建，直接部署 GitHub Pages

### 页面（6个）
1. **index.html** - 首页：Hero区域、个人简介、快速入口
2. **about.html** - 关于我：基本信息、工作经历、教育背景、专业技能
3. **files.html** - 文件共享：公开文件列表 + 私密入口
4. **private.html** - 私密文件：SHA-256密码验证（默认密码123456）
5. **gallery.html** - 图片画廊：分类过滤、灯箱大图预览
6. **contact.html** - 联系我：表单（mailto后备）+ 联系方式卡片

### 设计风格
- 蓝白配色，医疗专业风格
- 响应式设计，适配手机/平板/桌面
- 卡片式布局，SVG图标

### 文件结构
- css/style.css - 全局样式（CSS变量、响应式）
- js/main.js - 导航交互、滚动效果
- js/auth.js - SHA-256密码验证
- js/gallery.js - 画廊过滤和灯箱
- README.md - 使用指南
